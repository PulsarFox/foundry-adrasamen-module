# Phase 4: Quadralithe Equipment System - Redesigned Implementation

_Design Date: May 2, 2026_

## Overview

Complete redesign of the quadralithe system with enhanced functionality, formula-based effects, and deep integration with the existing mana, affinity, and spell systems. The system maintains the existing 2x2 grid UI while implementing robust D&D5e equipment integration and flexible configuration.

## Core Requirements Changes

### System Philosophy 
- **Formula-Based Everything**: All effects use configurable formulas instead of hardcoded values
- **D&D5e Native Integration**: Quadralithes are proper equipment items with custom configuration
- **Spell Integration**: Equipment provides spells via D&D5e cast activities
- **Zero Affinity Penalty**: Severe cost penalties for casting without required affinities
- **Enhanced Spellcasting**: Indirect characteristic linking for spell attack modifiers

## Spell Integration & Cost Calculation System

### Spellcasting Modifier Calculation (NEW)
```javascript
// New spellcasting modifier logic using indirect characteristic linking
function getSpellcastingModifier(actor, spell) {
  const affinityCosts = getSpellAffinityCosts(spell);
  
  // Find highest affinity cost used in the spell
  let highestCost = 0;
  let primaryAffinity = null;
  
  Object.entries(affinityCosts).forEach(([affinity, cost]) => {
    if (cost > highestCost) {
      highestCost = cost;
      primaryAffinity = affinity;
    }
  });
  
  if (!primaryAffinity) return 0;
  
  // Get characteristic linked to this affinity through primary/secondary system
  const affinityData = getAffinityData(actor);
  const characteristicLinking = getCharacteristicLinking(actor);
  
  let linkedCharacteristic = null;
  if (affinityData[primaryAffinity]?.isPrimary) {
    linkedCharacteristic = characteristicLinking.primary;
  } else if (affinityData[primaryAffinity]?.isSecondary) {
    linkedCharacteristic = characteristicLinking.secondary;
  }
  
  return linkedCharacteristic ? 
    actor.system.abilities[linkedCharacteristic]?.mod || 0 : 0;
}
```

### Enhanced Cost Reduction System (MODIFIED)
Extends existing `getCostReductions()` to include Nexus equipment effects:
```javascript
export function getCostReductions(actor) {
    if (!actor) return [];

    const reductions = [];

    // Existing affinity-based reductions from levels
    Object.values(AFFINITIES).forEach(affinityId => {
        const level = getAffinityLevel(actor, affinityId);
        if (level > 0) {
            reductions.push({
                type: "affinity",
                affinityId: affinityId,
                amount: level,
                origin: "level"
            });
        }
    });

    // NEW: Nexus quadralithe reductions (adds to ALL affinities)
    const equippedQuadralithes = actor.getFlag("adrasamen", "equippedQuadralithes") || {};
    if (equippedQuadralithes.nexus) {
        const nexusItem = actor.items.get(equippedQuadralithes.nexus);
        if (nexusItem) {
            const costReduction = evaluateFormula(
                nexusItem.system.quadralithe.effects.costReduction, 
                actor
            );
            if (costReduction !== 0) {
                // Add the same reduction to ALL affinities
                Object.values(AFFINITIES).forEach(affinityId => {
                    reductions.push({
                        type: "affinity",
                        affinityId: affinityId,
                        amount: costReduction,  // Can be negative!
                        origin: "nexus"
                    });
                });
            }
        }
    }

    return reductions;
}
```

### Zero Affinity Penalty System (NEW)
```javascript
// Enhanced spell cost calculation with zero affinity penalty
function calculateAffinityCostWithPenalty(actor, affinityId, baseCost) {
  if (baseCost <= 0) return 0;
  
  const affinityLevel = getAffinityLevel(actor, affinityId);
  let finalCost = baseCost;
  
  // Apply zero affinity penalty FIRST
  if (affinityLevel === 0) {
    const manaData = getManaData(actor);
    finalCost = baseCost + Math.floor(manaData.max / 2);  // Add penalty to base cost
  } else {
    // Normal reduction based on affinity level  
    finalCost = Math.max(0, baseCost - affinityLevel);
  }
  
  return finalCost;
}
```

## Formula System & Quadralithe Implementation

### Universal Formula Evaluator
```javascript
// Formula evaluation engine for all quadralithe effects
function evaluateFormula(formula, actor) {
  if (!formula || typeof formula !== 'string') return 0;
  
  // Simple number check
  const numValue = Number(formula);
  if (!isNaN(numValue)) return numValue;
  
  // Formula context variables
  const context = {
    // Actor abilities
    str: actor.system.abilities.str?.mod || 0,
    dex: actor.system.abilities.dex?.mod || 0,
    con: actor.system.abilities.con?.mod || 0,
    int: actor.system.abilities.int?.mod || 0,
    wis: actor.system.abilities.wis?.mod || 0,
    cha: actor.system.abilities.cha?.mod || 0,
    
    // Actor level
    level: actor.system.details.level || 1,
    
    // Affinity levels
    ...Object.fromEntries(
      Object.values(AFFINITIES).map(affinity => [
        affinity, getAffinityLevel(actor, affinity)
      ])
    ),
    
    // Special variables
    radiantBaseCost: 0  // Set by Radiant calculation
  };
  
  // Replace @variables with values and evaluate safely
  let processedFormula = formula;
  Object.entries(context).forEach(([key, value]) => {
    processedFormula = processedFormula.replace(
      new RegExp(`@${key}`, 'g'), 
      value
    );
  });
  
  try {
    return Function(`"use strict"; return (${processedFormula})`)();
  } catch (error) {
    console.warn(`Adrasamen | Invalid formula: ${formula}`, error);
    return 0;
  }
}
```

### Quadralithe Type Implementations

#### Morphos: Affinity Level Bonuses
- **Effect**: Formula-based bonuses to specific affinities
- **Integration**: Extends existing `getAffinityLevel()` function 
- **Examples**: `"1"`, `"@str + 1"`, `"@level / 2"`

#### Nexus: Mana & Cost Modifications  
- **Effect**: Formula-based max mana bonus + global cost reduction
- **Integration**: Modifies existing `affinityRecalculateMaxMana()` to replace hardcoded "4"
- **Cost System**: Uses existing affinity reduction system (9 reductions, one per affinity)
- **Examples**: Max mana `"4"`, cost reduction `"-1"` or `"@level"`

#### Radiant: Attack Roll Bonuses
- **Effect**: Base calculation + formula bonus for spell attacks
- **Base Calculation**: Lowest affinity level used by spell (0 if any affinity level is 0)
- **Formula Bonus**: Additive to base, can reference `@radiantBaseCost`
- **Examples**: Formula `"0"`, `"@radiantBaseCost"`, `"@radiantBaseCost * 2"`

#### Drain: Mana Generation Spells
- **Effect**: Creates 0-cost spell using D&D5e targeting system
- **Configuration**: Uses native RangeData and TargetData structures  
- **Integration**: Spell appears in activities when equipped
- **Default**: Range 9m, affects 1 object, generates configurable mana
